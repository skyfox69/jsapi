# Change log

## 0.2.0 ()

### Breaking changes

- The `schema` method no longer takes the `schema` keyword to refer another schema,
  for example `schema 'foo', schema: 'bar'`. Instead, the `ref` keyword can be used,
  for example `schema 'foo', ref: 'bar'`.
